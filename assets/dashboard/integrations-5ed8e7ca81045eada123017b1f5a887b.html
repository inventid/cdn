Stap 1:
Maak een Facebook event of pagina aan.
Stap 2:
Stuur de link van je Facebookpagina (bijvoorbeeld www.facebook.com/inventid.nl) door naar je account manager ({{ shop:event:organization:sales_manager.name }}) via {{ shop:event:organization:sales_manager.email_address }}.
Stap 3:
Je account manager stuurt je een link terug waarmee je binnen een paar klikken je verkoop aan je Facebookpagina hebt gekoppeld.

Het insluiten van het verkoopkanaal via een iframe:

<iframe
  src="{{ shop:url }}"
  width="770"
  height="600"
  style="border:none; margin-left:auto; margin-right:auto; width:770px;">
</iframe>
Let op!
Het iframe heeft een minimale breedte van 770px nodig om goed weergeven te kunnen worden. Zodra de betaling afgehandeld wordt zal men de website alsnog verlaten, gezien de betaling niet in een iframe afgehandeld mag worden wegens veiligheidseisen.
